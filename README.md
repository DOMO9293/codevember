## Running

Clone this repo, then:

```sh
npm install

# start day 1 demo
npm start 1

# start day 5 and launch browser
npm start -- 1 --open
```

You can `npm run start -- 5 --open` to launch day 5 and open the browser to [http://localhost:9966/](http://localhost:9966/).

To build all demos:

```sh
npm run build
```
